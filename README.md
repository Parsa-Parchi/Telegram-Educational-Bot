# Educational Telegram Bot

A backend-focused educational Telegram bot built with Python and PostgreSQL.

## Description

This project is a Telegram bot for managing educational services such as student question submission, registration flow, question bank access, and free educational content.

## Features

- Telegram bot menu using reply keyboards and inline buttons
- Student question submission workflow
- Admin and advisor role handling
- Registration and renewal flow
- Daily usage limits
- Student activation using activation codes
- Question bank access
- PostgreSQL database integration
- Channel membership verification
- Environment-based configuration using `.env`

## Technologies Used

- Python
- python-telegram-bot
- PostgreSQL
- psycopg
- python-dotenv
- Git / GitHub
